1. Create Array


    //constructor
    var myArr = new Array();
    myArr[0] = 'Happy';
    myArr[1] = 'New';
    myArr[2] = 'Year';

    > myArr;
      = ["Happy", "New", "Year"]

    > myArr.length;
      = 4

Additionally, you can set array length inside constructor. However, this is not recommended if you are not sure you will use the whole array. As this will create an array with "undefined" elements


    var myArr = new Array(100);
