there are three context in javaScript

  1. global context- default environment for code execution
  2. functional context-executing code inside a function
  3. eval context-inside eval execution
// global context
var myBar = 'my awesome variable';
var realBar = 'get drunk';

// function context
  var first = 'that',
      last = 'Dude';
  return first + ' js '+last;      
            
}

you will have as many function context as many function you have declared. function inside a function will create a function context inside the function. However, you will have only one global context.

each function call creates a new execution context event it calls itself
