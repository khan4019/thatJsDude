Getting Started with React

Have you ever planned to learn and master a new framework like React, but felt lack of time or motivation, even on the weekends?

Everyone knows learning alone is hard. In fact... It's enough to make a grown man cry!

Don’t feel sad! It’s not only you. Most of us are like this. Because our burning desire to become a React developer is slowed down by our laziness and unavoidable social events forced by our life partners. (Don’t tell the last line to your partner. Both you and I will be in trouble)

Regardless of the reasons, your skillsets are getting incompetent day by day. That’s why we created a daylong workshop so that you can invest a day to explore React and build a fully functional website with it.

Just give us one day . We will make you comfortable with React.

What You'll Learn

This workshop is broken into six modules to make your learning easy and effective.

Module-1: Getting Started with React
  • Understand the problems React solves
  • Writing very simple React App: Hello React
  • Introduction to JSX
Module-2: React Core Ideas
  • Virtual DOM
  • Component and Component Architecture
  • Inline Style or Loading CSS
Module-3: Thinking Like a React Developer
  • Component Life Cycle
  • Loading Data from Server
  • Events and Properties
Module-4: Integral Parts of a Large React App
  • Routing with React Router
  • Chart, Form, Grid
  • Optimize Productivity: Tooling, Code Editor
  • Little Touch on ES6, webpack, this, Babel
Module-5: Build a Professional React App
  • Large Project setup
  • Architecture of a Largest React App
  • Component Design and Interactions
  • Routing and Routing Control
Module-6: Deploy
  • Unit Test and Linting
  • Best practices
  • Build
  • Deploy
