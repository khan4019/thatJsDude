Getting Started with React

[Workshop] Getting Started with React

 

Description

React is the most popular JavaScript framework out there due to it’s clean and elegant approach to handle front end development. That’s why every front end developer should learn and start thinking like a React developer. Understanding React will help you to keep your skillset competitive, land the next job and feel confident at your work.

This workshop is specially designed for front end developers who want to learn and build application with React without any prior experience.

 

 

Who is this workshop for

Front end developers who want to upgrade their skillsets in order to get a raise or apply for better job under a better manager.

Lazy developers who are constantly hearing about react but still wondering what the heck is react.

Intermediate level Front end developers who are frustrated with the unorganized resources available on internet.

Beginners level programmer who are confused after seeing some tutorial and aren’t getting anyone to ask question and show a straight path to learn React.

Curious Angular developers who is pulling hair out to understand so many strict rules and limitations.

Back-end developers who are tired of sitting behind and want to come to lime light of front end world with the effective skill in a short period of time.

 

What You'll Learn

This workshop is broken into six modules to make your learning easy and effective.

 

Module-1: Getting Started with React

  • Understand the problems React solves
  • Writing very simple React App: Hello React
  • Intro to JSX

 

Module-2: React Core Ideas

  • More about JSX
  • Virtual DOM
  • Component and Component Oriented Architecture
  • Inline Style or Loading CSS

 

Module-3: Thinking Like React

  • Passing Dynamic Data
  • Properties
  • React Events
  • Component Life Cycle
  • Helper and Utility Functions
  • Loading Data from Server

 

Module-4: Integral Part of a Large React App

  • Routing with React Router
  • Charting in React
  • Grid in react
  • React Form
  • Talk about tooling
  • Code Editor
  • Little Touch on ES6

 

Module-5: Build a Professional React App

  • Use Starter Kit
  • Architecture of the React App
  • Add multiple Component
  • Component Interactions
  • Add chart

 

Module-6: Deploy

  • Build
  • Unit Test with Jest
  • Deploy
